Objectives

To improve knowledge about algorithms and programming languages.


To enhance problem-solving and mathematical analysis skills.


To provide a challenging and learning platform for future graduates
using various programming environments.

To define and modularize problems based on daily experiences
and solve them using coding.


To imbibe ethical, society driven coding mentality among
student coders.


To encourage students towards improving their awareness
of the current programming world, and increase organizing
and participation in coding challenges.


To empower students for learning various coding principles and
enable them to be industry ready
